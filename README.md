# HelpDesk AI Chatbot

## Project Description

HelpDesk AI is a customer support chatbot developed using Python and Streamlit. The chatbot provides automated responses to common customer queries such as password reset, pricing information, free trial availability, and contact support details.

## Features

* Interactive chat interface
* Customer support automation
* Quick question buttons
* Chat history management
* Streamlit web deployment

## Technologies Used

* Python
* Streamlit
* GitHub
